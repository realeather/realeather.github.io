windows搭建IIS

  • 2018-03-20

本次使用Windows Server 2008 R2安装IIS7,并在上面运行asp网站

  • 安装IIS
  • 配置IIS
  • 安装SQLServer,搭建asp网站

安装IIS


首先安装.net framework 4.7.21

点击屏幕左下角开始菜单右边的服务器图标打开服务器管理器

角色==>添加角色==>Web 服务器(IIS)

角色服务:勾选需要安装的功能,安装


安装完打开控制面板==>打开或关闭windows功能-勾选internet信息服务

配置IIS


浏览器访问虚拟机ip显示IIS7页面,可以删除应用程序池下的文件和网站目录下的Default Web Site


查看端口占用
netstat-ano

查看补丁
systeminfo

把aspcms_A5/ASPCMS273_UTF8目录里文件复制到C:/www目录

Internet信息服务==>网站==>添加网站==>绑定ip地址

访问网站,提示403禁止访问

Internet信息服务==>网站==>aspcms==>组或用户名==>Users权限选完全控制

Internet信息服务==>网站==>aspcms==>双击ASP==>启用父路径

Internet信息服务==>网站==>aspcms==>双击默认文档==>添加默认文档index.asp

Internet信息服务==>应用程序池==>编辑应用程序池 选择.NET Framework版本

编辑应用程序池==>高级设置==>启用32位应用程序true

安装SQLServer,搭建asp网站


安装SQLServer2008R2







连接到服务器登录SQLServer

新建数据库

新建表 编辑表


打开网站配置文件config/AspCms_Config 查看access数据库文件路径Const accessFilePath="data/#5Dp8Gh.asp 查看www/data目录


SQLServe导入access数据库文件,网站配置文件AspCms_Config中的修改Const dbType=1(0为access,1为sqlserver)

1. 安装工具


下载安装Node.js https://nodejs.org/en/

下载安装Visual Studio Code https://code.visualstudio.com/

下载安装github桌面版 https://desktop.github.com/

下载安装git for windows https://gitforwindows.org/

2. 安装hexo 配置本地文件


Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。官网有教程和文档 https://hexo.io/zh-cn/

先安装Node.js, 然后cmd执行命令安装hexo
npm install -g hexo-cli

新建文件夹blog(名字随意)存储hexo项目, 在blog目录下打开cmd命令窗口
hexo init 初始化hexo静态页面文件

用visual studio code打开blog文件夹,修改配置文件_config.yml

$ hexo new [layout] title
新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。例如hexo new "My First Post"

$ hexo clean
清除缓存文件 (db.json) 和已生成的静态文件 (public)。
在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。
$ hexo generate
生成静态文件。

$ hexo server
安装完成后，输入hexo server以启动服务器，您的网站会在 http://localhost:4000 下启动。在服务器启动期间，Hexo 会监视文件变动并自动更新，您无须重启服务器

浏览器访问http://localhost:4000

3. 更换hexo主题


官网寻找主题https://hexo.io/themes/
复制主题下载路径

blog/themes路径下打开cmd 执行命令下载主题(安装git for windows才能用git命令)
git clone 主题下载路径

编辑_confi.yml, theme值改成想使用的主题名

hexo clean 清除public网站文件
hexo generate 生成public网站文件
hexo server 开启本地服务

4. 同步本地文件至 github.io


注册登录github, 右上角New repository新建一个资料库


运行github desktop并登录
选择clone a repository(克隆资料库)

选择存储库，设置本地路径，点击clone

把hexo public目录下的文件复制到github项目路径下

打开github desktop输入描述后点击Commit to master提交修改
